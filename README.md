# ECRI Statistics Foundations

A series of lessons to build confidence in practical statistics for research.

<img src="/docs/readme_figures/imperial_ecri.png" width = 600>

This course [https://github.com/ImperialCollegeLondon/rcds_statistics_foundations](https://github.com/ImperialCollegeLondon/rcds_statistics_foundations) provides an introduction to the fields of statistical inference, probability theory, and hypothesis testing. The course is organized in eight sessions, covering the topics listed below:

- Descriptive statistics 11/3
- Foundations of probability (I) 18/3
- Foundations of probability (I) 25/3,
- Estimation and confidence  1/4
- Introduction to hypothesis testing (I) 15/4
- Introduction to hypothesis testing (II) 22/4 
- Modelling, dependency and correlation 29/4
- Introduction to Bayesian probability 6/5

### Key ideas

- *Fundamental* - should not aim to cover everything – focus on general principles and the most widely used methods. Emphasis on simulated sampling from a probability distribution as a unifying concept.
- *Applicable* - examples in code (R and python) for each skill, plus focused exercises. 
- *Accessible* - hybrid sessions to maximise access and encourage students to attend all sessions / catch up with recordings.
- *Integrated* - single gateway to teaching materials + code notebooks, same experience for R and python users. Materials should compile to a textbook format for easy reference. (Quarto?)

### Format

after each lesson. Weds? 3:30 – 5?
